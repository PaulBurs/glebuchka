Для создания ssh ключа в Git выполните следующие шаги:
1. Откройте терминал (или командную строку) на своем компьютере.
2. Введите команду ssh-keygen -t rsa -b 4096 -C "ваш_email@example.com", заменив "ваш_email@example.com" на свой адрес электронной почты.
3. Нажмите Enter, чтобы принять предложенное место для сохранения ключа (обычно это ~/.ssh/id_rsa).
4. Введите пароль, если хотите защитить свой ключ.
5. Выполните команду cat ~/.ssh/id_rsa.pub, чтобы скопировать содержимое публичного ключа.
6. Откройте настройки своего аккаунта на GitHub (или другом Git-сервисе) и найдите раздел "SSH and GPG keys".
7. Нажмите кнопку "New SSH key" (или аналогичную) и вставьте скопированное ранее содержимое публичного ключа.
8. Нажмите "Add SSH key" (или аналогичную) и следуйте инструкциям на экране для подтверждения добавления ключа.
Готово! Теперь вы можете использовать свой ssh ключ для доступа к Git-репозиториям без необходимости вводить пароль каждый раз.
